Requirement Type Status Difficulty Priority Detail
Desktop Operational Responsibility   Proposed Medium Medium
The DSM is ultimately responsible to ensure that each desktop is providing the correct environment for each worker to perform her tasks.
Manage packaging of software components   Proposed Medium Medium
Software must be delivered to the desktop in a package in compliance with the Standard Operating Environment Specification stipulated in the Technology Charter.
Constraint Type Status Detail
Must be managed within the budget Invariant Mandatory
There is a defined budget for desktop support. Unpredictable equipment or software failure must be managed within this budget.
