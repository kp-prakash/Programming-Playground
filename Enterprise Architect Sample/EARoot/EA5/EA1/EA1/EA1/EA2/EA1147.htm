In order to display the test definitions, in the diagram properties set
 -  [] Show Testing. 

 With standard elements such as Classes, Issues, Changes etc. the tests will then be displayed.

Testing for a UI definition is best defined using a Custom Test Case element.  The UI Elements do not show tests when the diagram is set to display tests.

To Show the tests against the Test Case element - right-click on the element and select from the context menu: Advanced Setting | Use Rectangle Notation.
