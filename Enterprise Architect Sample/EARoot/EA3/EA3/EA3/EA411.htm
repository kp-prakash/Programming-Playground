Scenario Type Detail
Create new Title Basic Path
1. User selects the "Create Title" command.
2. User selects a Publisher.
3. User enters Title, Author, Publisher, Cost, Markup, Description.
4. User presses Submit This.
5. Submitted details are validated.
6. New Title is created in the database.
7. Success message is displayed to User.
Create new title - Publisher not listed Alternate
1. User selects the "Create Title" command.
2. User tries to selects a Publisher - not found..
3. User Selects "Create Publisher" .
4. User enters Publisher details.
5. User enters Title, Author, Publisher, Cost, Markup, Description.
4. User presses the Submit button.
5. Submitted details are validated.
6. New Title is created in the database.
7. Success message is displayed to User.
