Scenario Type Detail
No entry Alternate
If the system does not find any entries against the user entry of Author, title, publisher then return a response to the user that the search is complete with no results.
Standard Search Basic Path
User can enter data to search by title, Author, Publisher or a combination of these entries.  The search result should return a list of published book defined within the system.
