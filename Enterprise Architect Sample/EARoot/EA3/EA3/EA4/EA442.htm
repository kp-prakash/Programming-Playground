Scenario Type Detail
Add Item Basic Path
This allows the user to add a item currently selected to the shopping basket.  On clicking on this the user is prompted for the quantity of this item that they want to add to the shopping basket.  The default quantity is one.  The key for the book and the quantity are stored in the shopping basket.
Add Item - zero entry Alternate
When the user to adds an item currently selected to the shopping basket they are prompted for the quantity of this item that they want to add to the shopping basket. If this is set to zero then no entry is made for this item.
