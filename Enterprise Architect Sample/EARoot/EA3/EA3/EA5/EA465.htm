Scenario Type Detail
Package Order Basic Path
1. Use case begins when Storeroom worker selects the "Package Order" command.
2. This displays the list of orders that have been processed.
3. User selects the order that has been packaged
4. User sets the status on this order to "Packaged"
5. Processing message is displayed to the User.
