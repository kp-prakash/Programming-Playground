Basic Path - Client
The use case starts when a Client selects the Close Account command from the Account Details Screen. The system will load the Client's account details and check whether the Client has any outstanding orders. If not, the Client is shown a Confirmation Dialog. If the Client replies yes, the account is marked as closed. Note that none of the details are removed from the system, for auditing purposes.
