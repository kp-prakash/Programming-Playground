JavaScript Primer

This is an introductory material, to learn JavaScript in an easy way. This is aimed at JavaScript beginners, though experienced developers can use this to refresh various concepts. In order to get complete use out of this tutorial, please do the following:

  • Open this html file using suitable editor.
  • Simultaneously open this html file using a browser of your choice.
  • Enable developer tools in the browser and navigate to the script tab (in IE) to view the script console.
  • This tutorial logs the messages to the script console.
  • Start reviewing the comments and code in a top-down fashion.
  • Execute the code and observe the behavior.
